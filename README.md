Download dependencies:
```
bundle install
```
Get Sass framework:
```
bourbon install
```
Build Sass + Angular.js Code:
```
ruby1.9.3 build.rb
```

Build JavaScript (ES6):
```
gulp
```

Deploy:
```
cd app
firefox default.htm
```
